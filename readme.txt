Display firecrackers on your WordPress site after activation.

== Description ==
The Firecracker Display plugin adds a visual effect of firecrackers to your entire website once activated. It creates a celebratory ambiance, enhancing the user experience for special occasions or events.

== Installation ==
1. Upload the `firecracker-display` folder to the `/wp-content/plugins/` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Firecrackers will now be displayed across your site.

== Frequently Asked Questions ==
= Can I customize the firecracker display? =
Currently, customization options are limited. However, you can modify the appearance and behavior of firecrackers by editing the 'firecracker.js' file in the plugin directory.
